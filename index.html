PVSio-web

PVSio-web is a new graphical tool to prototype and analyse user interface software. It is an animation environment that allows designers to load and simulate realistic user interfaces. Underneath, it uses SRI's state-of-the-art theorem prover PVS for analysis, and the PVS-io component as a basis for simulation .
